General Information


This tool is designed to show all Cloud Toll Free customers (H1 accounts) which are, or have been built in Platform28. This is the official repository to determine which ESSS IPM owns a specific Cloud Toll Free customer. Account assignments are managed by Karen Schlosser. For a customer to be added to this site, they must:
  • Have a signed contract for Cloud Toll Free services
  • Contacted Field IPM, who has already engaged EPI about Cloud Toll Free
ESSS can only be engaged for Cloud Toll Free implementations by EPI. Once engaged, Karen Schlosser assigns an IPM to the account.
Please see Zach Grauerholz if you run into issues with the site.

Build Status Definitions


Below is a description for each availible Build Status for Cloud Toll Free migrations. As accounts are added to the portal, the default status is "ESSS IPM Assigned". The ESSS IPM should update the status throughout the implementation process.
  1. ESSS IPM Assigned
    • EPI contacted ESSS and customer has a signed contract
    • Meetings to complete CSOW not started
  2. CSOW in Progress
    • Meeting with customer to complete CSOW & design
  3. Design Complete
    • CSOW complete and signed off on
    • In process of building in P28
  4. Implementation Complete
    • All routing in P28 built
    • Some TFNs might be taking traffic, but not all of them
  5. Migration Complete
    • All TFNs in P28 taking traffic
